**The ultimate Islamic companion for VS Code.** A professional reminder with a live countdown, aesthetic sidebar dashboard, daily Quran verses, and built-in tasbih.

---

## 🔥 Ultra Features

- 🕒 **Live HH:MM:SS Countdown** — Never miss a prayer with a real-time status bar clock.
- 🎨 **Aesthetic Sidebar Dashboard** — A beautiful view of today's schedule with prayer status cards.
- 📖 **Daily Quran Ayah** — Start your coding day with a beautiful random verse and translation.
- 📿 **Tasbih Counter** — Built-in dhikr counter right in your status bar. Click to increment!
- 🌙 **Ramadan Mode** — Auto-detects Ramadan, shows **Suhoor** and **Iftar** labels.
- 🔊 **Sound Alerts & Doa** — Notification sounds when it's time to pray, with a "Doa after Adzan" popup.
- 📍 **Auto-Location** — No setup required! Automatically detects your city via IP.
- 🌈 **Dynamic Colors** — Status bar reacts to upcoming prayers (Yellow for warning, Red for arrival).

---

## 🛠️ Commands

| Command | Description |
|---------|-------------|
| `Prayer Time: Show Schedule` | View today's full prayer schedule |
| `Prayer Time: Refresh` | Force sync with API |
| `Prayer Time: Increment Tasbih` | +1 to your Dhikr counter |
| `Prayer Time: Reset Tasbih` | Set Tasbih counter back to 0 |

---

## ⚙️ Configuration

- `prayer.playAdzanSound`: (boolean) Enable audio alerts.
- `prayer.dailyAyatEnabled`: (boolean) Show Quran verse on startup.
- `prayer.city`/`country`: Set manually or leave empty for auto-detect.
