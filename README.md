---

## ✨ Features

- � **Live HH:MM:SS Countdown** — Watch the time tick down every second in your status bar.
- 🌙 **Ramadan Mode** — Auto-detects Ramadan, shows **Suhoor** and **Iftar** labels, and Ramadan greetings.
- 📅 **Hijri Date** — Displays the current Hijri date right next to the countdown.
- � **Auto-Location** — No setup required! Automatically detects your city via IP if not configured.
- 🎨 **Smart UI Colors** — Status bar turns **Yellow** 10 minutes before prayer and **Red** when it's time!
- 🔔 **Popup Notifications** — Get alerted for every prayer, Suhoor, and Iftar.
- 📅 **Interactive Schedule** — Click the status bar to see today's full Hijri and prayer schedule.

---

## ⚙️ Configuration

| Setting | Default | Description |
|---------|---------|-------------|
| `prayer.city` | `""` | City name (leave empty for auto-detect). |
| `prayer.country` | `""` | Country name (leave empty for auto-detect). |
| `prayer.ramadanMode` | `true` | Enable Suhoor/Iftar labels. |
| `prayer.showHijriDate` | `true` | Show Hijri date in status bar. |
| `prayer.showSeconds` | `true` | Toggle HH:MM:SS vs HH:MM. |

---

## 🛠️ Commands

- `Prayer Time: Show Today's Schedule` (or click status bar)
- `Prayer Time: Refresh`
- `Prayer Time: Toggle Ramadan Mode`

---

## 📦 Installation

Just search for **"Prayer Time Reminder"** in the VS Code Extensions tab and hit install!
